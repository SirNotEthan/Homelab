# Skills

Skills are curated instruction files that teach the Home AI how to perform
repeatable tasks.

Each skill should be specific, reviewable, and useful without requiring hidden
context.

## Suggested structure

```text
skills/
  homelab/
  coding/
  research/
  personal/
```

## Skill format

Use Markdown. Start each skill with:

```markdown
# Skill name

## Purpose

## When to use

## Inputs

## Procedure

## Validation

## Notes
```

## Rule of thumb

The assistant can propose new skill drafts, but the user decides which skills
become trusted operating knowledge.
